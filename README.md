# 🏅 Notas dos Atletas

Este projeto foi desenvolvido em JavaScript e tem como objetivo calcular a média válida das notas de atletas em uma competição de ginástica artística.

## 📋 Regras da competição
- Cada atleta recebe **5 notas**.
- A **maior** e a **menor** nota são descartadas.
- A média é calculada com as **3 notas restantes**.

## 📥 Entrada de dados
Uma lista de atletas com nome e notas.

## 📤 Saída
O sistema exibe no console:
- Nome do atleta
- Notas obtidas
- Média válida calculada

## ▶️ Como executar
1. Copie o conteúdo do arquivo `notas-atletas.js`
2. Cole em um ambiente que execute JavaScript (como o DEVstart ou o console do navegador)
3. Veja o resultado no console

## 🚀 Tecnologias utilizadas
- JavaScript
