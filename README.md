# 🏅 Sistema de Avaliação de Atletas

Este projeto foi desenvolvido em JavaScript com o objetivo de calcular a média válida das notas de atletas em uma competição de ginástica artística, seguindo regras específicas de avaliação.

## 📋 Regras da competição
- Cada atleta recebe **5 notas**
- A **maior** e a **menor** nota são descartadas
- A média é calculada com as **3 notas restantes**

## 📥 Entrada de dados
Uma lista de atletas contendo:
- Nome
- Notas atribuídas pelos jurados

## 📤 Saída
O sistema exibe no console:
- Nome do atleta
- Todas as notas recebidas
- Média válida calculada

## ⚙️ Funcionalidades
- Ordenação das notas
- Remoção automática da maior e menor nota
- Cálculo da média com base nas regras da competição

## 🚀 Tecnologias utilizadas
- JavaScript

## 🎯 Objetivo do projeto
Projeto desenvolvido para praticar lógica de programação, manipulação de arrays e aplicação de regras de negócio em JavaScript.
